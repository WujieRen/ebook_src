# 下载并设置数据库连接器

Druid，Hive，Ranger，Oozie，和Superset要求有操作数据库。在安装过程中，您可以选择使用现有的数据库，或者让Ambari安装一个新的实例——对Hive而言。为了让Ambari能够连接到您选择的数据库，您必须在安装该组件之前直接从数据库供应商那里下载必要的数据库驱动和连接器。为了更好地准备您的安装和升级，请在您设置环境时设置数据库连接。
