# 登录Ambari Server

## 先决条件

Ambari Server必须处于正在运行状态。

使用web浏览器登录Ambari Web。

## 步骤

1. 将浏览器指向

   ```text
   http://<your.ambari.server>:8080
   ```

   ，其中<your.abmari.server>是您ambari server主机的主机名。

2. 使用默认的用户名/密码（`admin/admin`）登录Ambari Server。您可以在随后更改这些凭证。对于一个新的集群而言，集群安装向导会展示出一个欢迎页面。
